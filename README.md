# 智慧親師通 (Smart School LINE Bot)

這是一套專為補習班/安親班打造的「智慧打卡與家長通知系統」。整合了實體感應卡打卡、自動 Excel 報表結算，以及 LINE Bot 自動推播，協助補習班以最低的成本達成最高效率的數位化管理。

## 🌟 系統特色

1. **一鍵 Excel 同步**：免去繁瑣的建檔，只要將包含「學號」、「卡號」、「姓名」、「家長手機」的 Excel 檔案放入系統，按下一鍵即可完成數百人的資料庫建立與同步。
2. **防走失警示**：每日晚上定時巡邏，若學生有進班但未離班，將自動推播警示家長。
3. **每日出勤批次結算**：將同一家庭一整天的多筆打卡紀錄濃縮成「一封信」發送，極大化節省 LINE 官方帳號的推播費用（單日單家庭只消耗 1 個 Token）。
4. **營業安全控制**：打卡機具備「鎖定」與「解鎖」功能，避免非營業時段的誤刷。
5. **打烊自動結算**：每日結束營業時，系統自動將今日所有打卡紀錄匯出成一份精美的 Excel 報表供會計作帳。

## 💻 系統環境需求

- **作業系統**：Windows 10 / 11
- **Python**：Python 3.10 或以上版本
- **硬體需求**：一般的櫃台行政電腦即可（需具備網路連線與 USB 讀卡機/掃描槍）
- **環境準備**：需要一組 LINE Official Account (官方帳號) 的 `Channel Secret` 與 `Channel Access Token`。

## 🚀 第一次安裝與環境設定

1. **安裝 Python**：請至 Python 官網下載並安裝 Python (記得勾選 `Add Python to PATH`)。
2. **取得專案**：將此專案資料夾下載至櫃台電腦的桌面或 D 槽。
3. **配置金鑰**：
   在專案目錄下找到（或新建）一個名為 `.env` 的檔案，填入以下資訊：
   ```env
   LINE_CHANNEL_SECRET=您的_Channel_Secret
   LINE_CHANNEL_ACCESS_TOKEN=您的_Access_Token
   ```
4. **準備 Excel 名單**：
   請將學生名單命名為 `學生資料.xlsx` 並放入 `excels` 資料夾中。
   檔案必須包含以下欄位（大小寫須完全相符）：
   - `學號` (如: 112001)
   - `姓名` (如: 王小明)
   - `卡號` (讀卡機掃出的亂碼內碼)
   - `爸爸手機` 或 `媽媽手機` 或 `簡訊電話1` (必須為 09 開頭)

## 🎯 櫃台每日操作 SOP

這套系統的日常操作非常簡單，櫃台人員每天只需遵循以下三步：

1. **啟動系統**
   - 點擊桌面上的 **`啟動智慧親師通.bat`** 捷徑。
   - 系統會自動彈出兩個黑色視窗（伺服器與 Ngrok 連線），**請勿關閉這兩個黑視窗**，只要縮小即可。
   - 瀏覽器會自動彈出打卡畫面。

2. **開始營業 (解鎖)**
   - 點選畫面左上角的 **「▶️ 開始營業」**，此時打卡機輸入框才會發亮解鎖。
   - 學生即可拿感應卡靠近讀卡機「嗶」一聲完成打卡。

3. **結束營業 (結算)**
   - 打烊時，點選左上角的 **「⏹️ 結束營業並結算」**。
   - 系統會立刻鎖定打卡機（避免誤刷），並自動下載一份 `YYYY-MM-DD_出勤紀錄.xlsx` 到您的電腦中。
   - 該報表同時會永久備份在專案的 `excels/Students/` 目錄下。
   - 確認下載完成後，即可關閉網頁與黑色視窗，安心下班！
